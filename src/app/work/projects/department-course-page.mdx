## Overview

A comprehensive web platform designed to provide detailed information about postgraduate courses offered by the Computer Engineering Department at the University of Peradeniya. The system addresses the need for a centralized, informative platform for course management and student enrollment.

## Key Features

- **Course Catalog**: Comprehensive listing of all postgraduate courses with detailed information
- **Course Categories**: Organized by Core Courses, Technical Electives, and Research Topics
- **Detailed Course Pages**: Individual pages for each course with comprehensive information
- **Registration System**: Google Forms integration for course registration
- **Academic Staff Integration**: Links to academic staff and course coordinators
- **Responsive Design**: Mobile-friendly interface for accessibility

## Technical Implementation

- **Frontend**: HTML5, CSS3, JavaScript for modern web development
- **Static Site Generator**: Jekyll for efficient static site generation
- **Data Management**: JSON for structured course data storage
- **Hosting**: GitHub Pages for reliable and cost-effective hosting
- **Form Integration**: Google Forms for seamless registration process
- **Version Control**: Git for collaborative development

## Course Information System

- **Course Details**: Comprehensive information including descriptions, prerequisites, and learning outcomes
- **Academic Information**: Course coordinators, lecturers, and contact information
- **Grading System**: Grade distribution and assessment criteria
- **Reference Materials**: Course books and additional resources
- **Timetable Integration**: Lecture, lab, and tutorial hour breakdowns

## User Experience Features

- **Navigation**: Intuitive navigation with Home, All Courses, and Staff sections
- **Search Functionality**: Easy course discovery and filtering
- **Responsive Layout**: Optimized for desktop, tablet, and mobile devices
- **Accessibility**: WCAG compliant design for inclusive access
- **Performance**: Fast loading times with optimized static assets

## Key Achievements

- **Information Centralization**: Created a single source of truth for course information
- **User-Friendly Interface**: Intuitive design for easy navigation and course discovery
- **Academic Integration**: Seamless integration with existing academic systems
- **Scalability**: Jekyll-based architecture supports easy content updates
- **Cost Efficiency**: GitHub Pages hosting eliminates infrastructure costs

## Challenges and Solutions

The main challenge was creating a system that could handle the diverse information requirements for different course types while maintaining consistency. The solution involved developing a flexible JSON-based data structure and implementing responsive design principles. Additionally, ensuring seamless integration with existing academic workflows required careful consideration of user needs and system requirements.

## Outcome

The Department Course Page system successfully provides a comprehensive platform for postgraduate course information management. The system has improved course discoverability and streamlined the enrollment process, benefiting both students and academic staff. The Jekyll-based architecture ensures easy maintenance and content updates.
